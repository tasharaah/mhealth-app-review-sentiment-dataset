# Dataset use notes

This repository includes a cleaned dataset of publicly available user reviews from app-store pages. The dataset is provided for academic and non-commercial research on digital health, sentiment analysis, and NLP.

## Responsible-use expectations

- Use the data only for research, education, and reproducibility.
- Do not attempt to identify, contact, or profile individual reviewers.
- Do not use the dataset for decisions about individual users.
- Do not redistribute modified versions without clearly documenting the changes.
- Users are responsible for ensuring that their use complies with relevant platform terms, institutional policies, and local regulations.

## Privacy note

The released dataset does not intentionally include usernames or reviewer identifiers. However, user-generated review text may occasionally contain incidental personal information. Users should handle the dataset responsibly and avoid publishing individual review text examples that contain sensitive or identifying details.

## Citation

Please cite the associated manuscript and this repository if you use the dataset.
