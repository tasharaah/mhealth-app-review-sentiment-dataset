# External validation data

The manuscript reports external validation on reserved Bearable Google Play reviews. This folder is reserved for that file if it is released.

Recommended filename:

```text
Bearable_GooglePlay_External_Validation.csv
```

The external validation file should use the same schema as the main dataset when possible:

```text
source,review_description,review_date,rating,app_name,sentiment_value
```

If the external validation data are not released, update the manuscript data availability statement accordingly.
